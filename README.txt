# Gemini Mesh PoC Quickstart

## Files
- `gemini_test.py`: Python script to test Gemini via Vertex AI
- `run.sh`: Shell script to execute the test
- `gemini-mesh-agent-key.json`: Your GCP service account key (upload separately)

## Instructions
1. Upload `gemini_test.py` and `run.sh` to Google Cloud Shell.
2. Also upload your `gemini-mesh-agent-key.json` file.
3. Run:
   chmod +x run.sh
   ./run.sh
